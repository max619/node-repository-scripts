# node-repository-scripts

This package aims to solve problem of managing a lot of node scripts and also provide an easy way to execute scripts without need of creating cli tools.

This project is intended to be a successor of [nps](https://github.com/sezna/nps), so if you are thinking about migration to `node-repository-scripts` and found some features missing, consider leaving an issue.

## Features

- TypeScript support
- Concurrent scripts execution
- Help generation
